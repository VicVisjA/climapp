# # 🌦️ App del Clima

Una aplicación web moderna que permite consultar el clima en tiempo real de cualquier ciudad, mostrando información detallada y un pronóstico extendido de 5 días.

---

## ✨ Características

* 🌍 Búsqueda de cualquier ciudad del mundo
* 🌡️ Temperatura actual y sensación térmica
* 💧 Nivel de humedad
* 🌬️ Velocidad del viento
* 🌅 Hora de amanecer y atardecer
* 🌧️ Probabilidad de lluvia
* 📅 Pronóstico de 5 días
* 🎨 Interfaz moderna y responsive

---

## 🛠️ Tecnologías utilizadas

* ⚡ Angular (Standalone Components)
* 🎨 HTML + CSS
* 🌐 API de clima (como OpenWeatherMap u otra)
* 🚀 GitHub Pages para despliegue

---

## 📦 Instalación

Clona el repositorio:

```bash
git clone https://github.com/TU-USUARIO/TU-REPO.git
cd TU-REPO
```

Instala dependencias:

```bash
npm install
```

Ejecuta el proyecto:

```bash
ng serve
```

Abre en tu navegador:

```
http://localhost:4200
```

---

## 🌐 Despliegue en GitHub Pages

```bash
ng build --base-href="/TU-REPO/"
npx angular-cli-ghpages --dir=dist/TU-PROYECTO/browser
```

## ⚙️ Configuración

Si usas una API de clima, asegúrate de agregar tu API KEY en el servicio correspondiente:

```ts
apiKey = 'TU_API_KEY';
```
